#  AI Vision: Face vs. Hand Classifier

A real-time image classification web application built with **TensorFlow.js** and **Google Teachable Machine**. 

##  Overview
This project demonstrates the power of **Edge AI** by running a neural network directly in the browser. The model was trained using Transfer Learning to distinguish between a user's face and their hand with high confidence.

###  Features
- **Real-time Detection:** Zero-latency classification using the user's webcam.
- **Modern UI:** Dark-mode dashboard with animated scanning effects and dynamic progress bars.
- **Privacy Focused:** All processing happens locally on the client's device; no images are sent to a server.

##  Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Properties & Animations), JavaScript (ES6+).
- **Machine Learning:** TensorFlow.js, Teachable Machine Image Library.
- **Hosting:** GitHub Pages.

## How to Run Locally
1. Clone the repository: `git clone https://github.com/YOUR_USERNAME/AI-Image-Classifier.git`
2. Open the folder in VS Code.
3. Run using the **Live Server** extension to bypass CORS browser security.
